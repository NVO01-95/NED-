Help & Support

This page is for application support, bug reports and feature requests. For operational phrases, go to Communication.

Before you contact support

  • Make sure you are using the latest saved data (refresh the page).
  • If a feature behaves strangely, try logging out and logging in again.
  • For route issues, double-check waypoint format: lat, lon or saved location names (Admin Locations).

Report a bug

Please include as much detail as possible so we can reproduce the problem.

Include:

  • What you tried to do (steps).
  • What you expected to happen.
  • What actually happened (error message / screenshot).
  • Your browser and OS (Windows / Linux / mobile).
  • The date & time (local).

Tip: if you see a Flask/traceback error, copy the first line and the last few lines.

Feature request

NED is evolving. If you want a feature, describe the use-case (real scenario) and what outcome you need.

Examples:

  • “I want route suggestions for inland navigation segments.”
  • “I want an admin-only log of user actions.”
  • “I want emergency phrase packs per region/language.”
